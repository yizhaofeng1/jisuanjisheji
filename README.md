  这个仓库存放的是我计算机设计大赛的作品，当时获得湖北省省一和决赛国三。

  由于当时暑假有其他事情，所以其实省赛拿到省一后虽然上推了国赛，但其实只是做了个PPT，作品并没有任何改良，所以还是有一些提升空间的。
根据评委说：
- 地图可以换成山海经的地图体现分布更好
- 可以做一下3D的异兽模型
- 我们有些地图忘记标注出处了，虽然都是pyecharts上面的，但是还是要都标注出来 

我本人由于一开始只是为了做数据可视化的展示，所以评委说的这些点其实我一开始就没考虑就是了。 

不过，我自己要是接着做可能就是将中国地图上每个省份再附上当地的和异兽有关的景点链接来体现实际价值就是了。 

注意：由于本人和团队当时对前端代码的能力都不高，所以前端代码写的很简单，而且许多图都是我直接用pyechasrt库直接做的，本质上我没有写JS代码，当然我本人也不会，我本人主要是在后端上，Django服务器的书写。 
但是当时确实也是边学边写，其实Djanggo用的肯定只为了能展示效果，路由那方便写的相当随意，所以这个作品各位就仅供参考了！要是觉得不行也不要喷。 

不过这个比赛，我给的建议就是创意度更加重要，其实当时我们赛道上没有使用山海经作为主题的，所以设计设计么，创意度肯定第一呀！ 
要是这个分享对你有帮助，就点个Satrt吧！
